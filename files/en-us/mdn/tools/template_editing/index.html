On MDN, templates written in KumaScript are used to automate content generation and customization within pages. Each template is a separate file under the macros directory of the KumaScript GitHub repository.

Anyone editing MDN wiki pages can invoke templates via macros in MDN articles.  Anyone can create and edit templates via the KumaScript GitHub repository using standard open-source practices (fork the repo, create a branch, make your changes, and submit a pull-request for review). Note that submitting a pull request is currently the only way to update translated strings in templates that contain them.
